# Cg-Graphs

This repository is responsible for graphs and charts for Cybergreen Project

## Setup

To install all dependencies:
```
$ npm install
```

To run local dev server:
```
$ npm start
```

To build production bundle.js:
```
$ npm run build
```

## Tests

To run all tests
```
$ npm test
```
